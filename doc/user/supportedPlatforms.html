OpenCOR can be used on the following versions of Windows, Linux and OS X. Alternatively, you can try to build it yourself.

Windows
  • Up to OpenCOR 0.3: supported on both the 32-bit and 64-bit versions of Windows XP and later.
  • Future versions of OpenCOR: supported on both the 32-bit and 64-bit versions of Windows 7 and later.
Linux
  • OpenCOR 0.1.x and 0.2: supported on both the 32-bit and 64-bit versions of Ubuntu 12.04 LTS (Precise Pangolin) and later.
  • OpenCOR 0.3: supported on both the 32-bit and 64-bit versions of Ubuntu 14.04 LTS (Trusty Tahr) and later.

in all cases, OpenCOR may also work with earlier versions of Ubuntu, as well as with other Linux distributions, but additional system libraries may be needed in the latter case.

OS X
  • OpenCOR 0.1.x: supported on OS X 10.8 (Mountain Lion) and later.
  • OpenCOR 0.2 and later: supported on Mac OS X 10.7 (Lion) and later.
