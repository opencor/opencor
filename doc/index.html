OpenCOR

OpenCOR stands for Open CO[omputational] physiology R[esource]. It is a cross-platform CellML-based modelling environment which has, from its inception, been developed with modellers, experimentalists and teachers/students in mind. It can be used to organise, edit, simulate and analyse CellML files.
