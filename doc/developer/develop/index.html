As is often the case when it comes to software development, the best way to get familiar with a probject is by having a look at its source code.

OpenCOR namespace

As you will see, all of OpenCOR's code is and should be made available under the OpenCOR namespace (e.g. [OpenCOR]/src/mainwindow.cpp and [OpenCOR]/src/mainwindow.h). The only exception to this rule is [OpenCOR]/src/main.cpp where OpenCOR's main function uses the OpenCOR namespace.
