Third-party Libraries

OpenCOR uses both software- and image-based third-party libraries, all of which are listed below, together with the type of license under which they are released.

In all cases, only the files actually required by OpenCOR have been included in our Git repository. In the case of a software library, a/the CMakeLists.txt file will have been created/modified so that the library could be built as part of our build process.

Software
  • QtSingleApplication: a component which ensures that only one instance of OpenCOR can be run at once.
    • Location: [OpenCOR]/3rdparty/QtSingleApplication
    • License: BSD license
Image
  • Printable world flags: the two (British and French) flag icons used in the menu to specify the locale to be used by OpenCOR.
    • Location: [OpenCOR]/res
    • License: free to use, no license
  • Boomy multiplatform: the remaining icons used in OpenCOR.
    • Location: [OpenCOR]/res
    • License: free to use, no license
