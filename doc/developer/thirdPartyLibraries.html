OpenCOR uses various APIs, as well as software- and image-based third-party libraries, all of which are listed below, together with the type of license(s) under which they are released, if any.

APIs
  • ﻿ libBioSignalML: a library for reading and writing BioSignalML documents.
    • Location: [OpenCOR]/src/plugins/api/BioSignalMLAPI.
    • Licensing: none.
  • ﻿ CellML API (from opencor/cellml-api on GitHub): an interface to manipulate and process CellML documents.
    • Location: [OpenCOR]/src/plugins/api/CellMLAPI.
    • Licensing: MPL v1.1, GPL v2.0 (or later) and LGPL v2.1 (or later).
  • ﻿ libSBML 5.15.0: a programming library to help you read, write, manipulate, translate and validate SBML files and data streams.
    • Location: [OpenCOR]/src/plugins/api/SBMLAPI.
    • Licensing: LGPL v2.1 (or later).
  • ﻿ libSEDML 0.3.1: a library for reading and writing SED-ML models.
    • Location: [OpenCOR]/src/plugins/api/SEDMLAPI.
    • Licensing: BSD 2-Clause.
Software
  • diff-match-patch: a set of libraries that offer robust algorithms to perform the operations required for synchronising plain text.
    • Location: [OpenCOR]/src/3rdparty/diff_match_patch.
    • Licensing: Apache v2.0.

      we use the STL version of that code.

  • Google Code Prettify: a JavaScript module and CSS file for syntax highlighting of source code snippets in an HTML page.
    • Location: [OpenCOR]/doc/3rdparty/googleCodePrettify.
    • Licensing: Apache v2.0.
  • jQuery 3.2.1: a fast and concise JavaScript library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.
    • Location: [OpenCOR]/doc/3rdparty/jQuery.
    • Licensing: MIT.
  • KaTeX 0.7.1: fast math typesetting for the web.
    • Location: [OpenCOR]/doc/3rdparty/KaTeX.
    • Licensing: MIT.
  • libgit2 0.26: a portable, pure C implementation of the Git core methods provided as a re-entrant linkable library with a solid API.
    • Location: [OpenCOR]/src/plugins/thirdParty/libgit2.
    • Licensing: GPL v2.0 with linking exception.
  • LibXDiff 0.23: a library that implements basic and yet complete functionalities to create file differences/patches to both binary and text files.
    • Location: [OpenCOR]/src/plugins/thirdParty/LibXDiff.
    • Licensing: LGPL v2.1.
  • LLVM+Clang 4.0.0: a collection of modular and reusable compiler and toolchain technologies.
    • Location: [OpenCOR]/src/plugins/thirdParty/LLVMClang.
    • Licensing: UoI/NCSA.
  • OAuth 2.0 for Qt 1.0.1: a library that encapsulates the OAuth 1.0 and 2.0 client authentication flows, and the sending of authenticated HTTP requests.
    • Location: [OpenCOR]/src/plugins/thirdParty/OAuth.
    • Licensing: BSD 2-Clause.
  • OpenSSL 1.0.2l: a toolkit implementing the Secure Sockets Layer (SSL v2/v3) and Transport Layer Security (TLS) protocols, as well as a full-strength general purpose cryptography library.
    • Location: [OpenCOR]/src/plugins/thirdParty/OpenSSL.
    • Licensing: OpenSSL and SSLeay, which are both BSD-style licenses (more information here).
  • QScintilla 2.10: a Qt port of the Scintilla editing component.
    • Location: [OpenCOR]/src/plugins/thirdParty/QScintilla.
    • Licensing: GPL v3.0.
  • QtSingleApplication: a class that ensures that only one instance of the GUI version of OpenCOR can be run at any given time.
    • Location: [OpenCOR]/src/3rdparty/QtSingleApplication.
    • Licensing: LGPL v2.1 (or later).
  • QtWebKit: a Qt port of WebKit, a web browser engine.
    • Location: [OpenCOR]/src/3rdparty/QtWebKit.
    • Licensing: GPL v2.0 (or later) and LGPL v2.1 (or later).
  • Qwt 6.1.3: a set of widgets and utility classes primarily aimed at programs with a technical background.
    • Location: [OpenCOR]/src/plugins/thirdParty/Qwt.
    • Licensing: Qwt License v1.0 (a slightly less restricive version of LGPL v2.1) and LGPL v2.1 for the MathML text engine, which is based on the QtMmlWidget code.

      we have contributed some code to the MathML renderer, but it has yet to be fully incorporated in the Qwt project. So, in the meantime, we use our version of the MathML renderer.

  • SUNDIALS 2.7.0: a SUite of Non-linear and DIfferential/ALgebraic equation Solvers.
    • Location: [OpenCOR]/src/plugins/thirdParty/SUNDIALS.
    • Licensing: BSD 3-Clause.
  • web-xslt: XSLT and JavaScript code intended mostly for manipulating MathML and OpenMath.
    • Location: [OpenCOR]/src/plugins/miscellaneous/Core/res/web-xslt.
    • Licensing: W3C, Apache v2.0, MIT, MPL v1.1 or MPL v2.0.

      we currently use only (a slightly improved version of) ctopff.xsl to convert Content MathML to Presentation MathML.

  • ﻿ Zinc 3.1.1: a library for creating software to interact with and visualise complex finite element models and image-based fields.
    • Location: [OpenCOR]/src/plugins/thirdParty/Zinc.
    • Licensing: MPL v2.0.
  • zlib 1.2.11: a massively spiffy yet delicately unobtrusive compression library.
    • Location: [OpenCOR]/src/plugins/thirdParty/zlib.
    • Licensing: Zlib.
Images

    • Licensing: LGPL v3.0.

    • Location: [OpenCOR]/src/plugins/widget/EditorWidget/res/qtCreator.
    • Licensing: GPL v3.0.
