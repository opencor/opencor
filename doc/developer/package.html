Package

Packaging of OpenCOR can only be done from the command line, using [OpenCOR]/package[.bat]. Note that though the Linux package is originally meant for Ubuntu 12.04 LTS (Precise Pangolin; both 32-bit and 64-bit), it should also work with other versions of Ubuntu, as well as other Linux distributions though additional libraries may be required in this latter case.
